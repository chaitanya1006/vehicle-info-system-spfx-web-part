## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - `npm install -g @rushstack/heft`
  - `npm install`
  - `heft start`


- **Task #1:** Fetch the logged-in user profile from Azure tenant using Microsoft Graph API and display in web part
- **Task #2:** Handle CRUD (ADD / READ / UPDATE / DELETE) operations on SharePoint List 
- **Task #3:** Access the Azure Function on behalf of logged-in user by using **AadHttpClient**. **The Azure Function is protected by Entra ID**.
